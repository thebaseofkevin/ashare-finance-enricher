# 利润表

这里是 stock.financials 所有字段的全面、完整翻译对照表：

| 英文科目 | 中文含义 |
| --- | --- |
| Total Revenue | 总营收 |
| Operating Revenue | 营业收入 |
| Cost Of Revenue | 营业成本 |
| Gross Profit | 毛利 |
| Operating Expense | 营业费用 |
| Selling General And Administration | 销售、一般及管理费用 (SG&A) |
| Selling And Marketing Expense | 销售与营销费用 |
| General And Administrative Expense | 一般及行政费用 |
| Research And Development | 研发费用 |
| Depreciation And Amortization In Income Statement | 损益表中的折旧与摊销 |
| Depreciation Income Statement | 折旧（损益表） |
| Amortization | 摊销 |
| Rent And Landing Fees | 租金及着陆费 |
| Other Operating Expenses | 其他营业费用 |
| Operating Income | 营业利润 |
| Total Operating Income As Reported | 报告的营业利润 |
| Net Non Operating Interest Income Expense | 净非营业利息收支 |
| Interest Income Non Operating | 非营业利息收入 |
| Interest Expense Non Operating | 非营业利息支出 |
| Total Other Finance Cost | 其他融资成本总额 |
| Special Income Charges | 特别收支/特殊项目 |
| Write Off | 资产减值冲销 |
| Impairment Of Capital Assets | 资本资产减值 |
| Other Special Charges | 其他特别费用 |
| Other Non Operating Income Expenses | 其他非营业收支 |
| Pretax Income | 税前利润 |
| Tax Provision | 所得税拨备 |
| Net Income Continuous Operations | 持续经营净利润 |
| Net Income Including Noncontrolling Interests | 包含少数股东权益的净利润 |
| Minority Interests | 少数股东权益 |
| Net Income | 净利润 |
| Otherunder Preferred Stock Dividend | 优先股股利调整（其他） |
| Net Income Common Stockholders | 归属于普通股股东的净利润 |
| Basic EPS | 基本每股收益 |
| Diluted EPS | 稀释每股收益 |
| Basic Average Shares | 基本平均股数 |
| Diluted Average Shares | 稀释平均股数 |
| Rent Expense Supplemental | 租金费用补充 |
| Total Expenses | 费用总额 |
| Net Income From Continuing And Discontinued Operation | 持续和终止经营净利润 |
| Normalized Income | 正常化净利润 |
| Interest Income | 利息收入 |
| Interest Expense | 利息支出 |
| Net Interest Income | 净利息收入 |
| EBIT | 息税前利润 (Earnings Before Interest and Taxes) |
| EBITDA | 息税折旧摊销前利润 (Earnings Before Interest, Taxes, Depreciation, and Amortization) |
| Reconciled Cost Of Revenue | 调整后的营业成本 |
| Reconciled Depreciation | 调整后的折旧 |
| Net Income From Continuing Operation Net Minority Interest | 扣除少数股东权益后的持续经营净利润 |
| Total Unusual Items Excluding Goodwill | 扣除商誉的特殊项目总额 |
| Total Unusual Items | 特殊项目总额 |
| Normalized EBITDA | 正常化 EBITDA |
| Tax Rate For Calcs | 用于计算的税率 |
| Tax Effect Of Unusual Items | 特殊项目的税收影响 |
