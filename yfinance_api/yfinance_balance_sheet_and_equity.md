# 资产负债表和股东权益表

这是 yfinance 中资产负债表（Balance Sheet）的科目映射列表。以下是该列表的中文翻译对照表：

| 英文科目 | 中文含义 |
| --- | --- |
| Total Assets | 总资产 |
| Current Assets | 流动资产 |
| Cash Cash Equivalents And Short Term Investments | 现金、现金等价物及短期投资 |
| Cash And Cash Equivalents | 现金及现金等价物 |
| Cash Equivalents | 现金等价物 |
| Cash Financial | 货币资金 |
| Gross Accounts Receivable | 应收账款总额 |
| Allowance For Doubtful Accounts Receivable | 应收账款坏账准备 |
| Accounts Receivable | 应收账款 |
| Other Receivables | 其他应收款 |
| Inventory | 存货 |
| Raw Materials | 原材料 |
| Work In Process | 在产品 |
| Finished Goods | 产成品 |
| Prepaid Assets | 预付款项 |
| Other Current Assets | 其他流动资产 |
| Total Non Current Assets | 非流动资产总计 |
| Gross PPE | 固定资产原值 (Property, Plant & Equipment) |
| Accumulated Depreciation | 累计折旧 |
| Net PPE | 固定资产净值 |
| Construction In Progress | 在建工程 |
| Properties | 房地产/不动产 |
| Machinery Furniture Equipment | 机器、家具及设备 |
| Other Properties | 其他房地产 |
| Goodwill And Other Intangible Assets | 商誉及其他无形资产 |
| Other Intangible Assets | 其他无形资产 |
| Long Term Equity Investment | 长期股权投资 |
| Non Current Deferred Taxes Assets | 非流动递延所得税资产 |
| Non Current Deferred Assets | 非流动递延资产 |
| Other Non Current Assets | 其他非流动资产 |
| Total Liabilities Net Minority Interest | 负债合计（净少数股东权益） |
| Current Liabilities | 流动负债 |
| Accounts Payable | 应付账款 |
| Total Tax Payable | 应交税费 |
| Payables | 应付款项 |
| Other Payable | 其他应付款 |
| Current Debt And Capital Lease Obligation | 流动债务及资本租赁义务 |
| Other Current Liabilities | 其他流动负债 |
| Total Non Current Liabilities Net Minority Interest | 非流动负债总计（净少数股东权益） |
| Long Term Debt | 长期债务 |
| Long Term Debt And Capital Lease Obligation | 长期债务及资本租赁义务 |
| Capital Lease Obligations | 资本租赁义务 |
| Non Current Deferred Taxes Liabilities | 非流动递延所得税负债 |
| Non Current Deferred Revenue | 非流动递延收益 |
| Total Equity Gross Minority Interest | 股东权益合计（含少数股东权益） |
| Stockholders Equity | 股东权益 |
| Common Stock Equity | 普通股股东权益 |
| Minority Interest | 少数股东权益 |
| Capital Stock | 股本 |
| Common Stock | 普通股 |
| Additional Paid In Capital | 资本公积 |
| Retained Earnings | 盈余公积/留存收益 |
| Net Tangible Assets | 净有形资产 |
| Working Capital | 营运资金 |
| Invested Capital | 投入资本 |
| Tangible Book Value | 有形账面价值 |
| Total Debt | 总债务 |
| Total Capitalization | 总资本化 |
| Share Issued | 已发行股份 |
| Ordinary Shares Number | 普通股股数 |
